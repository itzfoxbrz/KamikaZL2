Server Commands
     1. .premium Your premium status can be checked.
     2. .deposit Converts 1,000,000,000 Adena to 1 Goldbar
     3. .withdraw Converts 1 Goldbar to 1,000,000,000 Adena
     4. .apon/.potionon Enables automatic use of potions when below 70%.
     5. .apoff/potionoff  Disables automatic use of potions when below 70%.
